Projects

Pac-Man AI

Using Python and the libary PyGame, I recreated the game of Pac-Man. After I had the game created and functional, I hooked up the controls of Pac-Man to a neural network that used the NeuroEvolution of Augmenting Topologies (NEAT) algorithm.

This was a great learning experience that required both pathfinding algorithms for the ghosts' movement and machine learning concepts and methodologies to implement the neural network.

Flappy Bird in JavaScript

Using vanilla JavaScript and a little bit of HTML, I recreated the game of Flappy Bird. The game keeps track of your current score as well as high score for your current session.

This was a fun project that gave me some good exposure to JavaScript and its potential for interactivity in other applications.

Music Store/Database

In this project, I created a website that allowed users to browse and "purchase" music using a database found online of a music store (songs, employees, customers. and others). There was also an employee login so that they could see the customers they have sold to and metrics pertaining to their performance.

I learned a lot about interacting with a database via SQL in this project as well as how a database should be structured.
